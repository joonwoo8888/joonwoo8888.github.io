# OKD란?

OKD는 지속적인 응용 프로그램 개발 및 배포에 최적화 된 Kubernetes 배포판입니다. OKD는 Kubernetes 위에 개발자 및 운영 중심 도구를 추가 하여 빠른 응용 프로그램 개발, 쉬운 배포 및 확장, 중소 규모 팀의 장기 수명주기 유지 관리를 지원합니다.

OKD Install, WEB Console, App Build and Deploy
